# Banana Shop 🍌

A beginner ecommerce checkout prototype built with Node.js and Express.

## Features
- Product quantity selection
- Live order summary
- Fake payment form
- Checkout API
- Success page redirect

## Tech Stack
- Node.js
- Express
- HTML/CSS/JavaScript

## Install
npm install

## Run
node server.js

Visit:
http://localhost:3000

## Disclaimer
Demo project only.
Do NOT enter real payment information.
No real payment processing is implemented.
